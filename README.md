# 통상임금 계산기

한국 노동법의 통상임금 계산을 돕는 간단한 CLI 도구입니다.

2024년 대법원 판례에 따라 "고정성 징표"를 별도 요건으로 보지 않고, 지급 형태와 목적을 중심으로 통상임금 포함 여부를 판단합니다.

## 사용 방법

1. `python 통상임금계산기.py`를 실행합니다.
2. 기본급과 수당 내역을 입력합니다.
3. 각 항목의 통상임금 포함 여부와 통상임금 산식 결과를 확인합니다.

## 특징

- 기본급과 정기적 수당을 통상임금 산정에 포함
- 업무 성격이 명백히 성과급인 요소는 제외
- 2024년 판례 취지를 반영하여 고정성 징표를 별도 판정 요소로 두지 않음
- 일당/시급 환산 결과 제공
