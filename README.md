# 퇴직금 산정 프로그램

한국 노동법에 따른 퇴직금 계산을 돕는 간단한 CLI 도구입니다.

퇴직금 = 평균임금 × 근속연수 × 30일

- 평균임금: 퇴사 전 3개월 임금 총액 ÷ 90일
- 근속연수: 입사일부터 퇴사일까지의 기간 (년 단위)

## 사용 방법

1. `python 퇴직금계산기.py`를 실행합니다.
2. 입사일, 퇴사일, 최근 3개월 임금을 입력합니다.
3. 퇴직금 산정 결과를 확인합니다.

## 특징

- 입사일과 퇴사일로부터 근속연수 자동 계산
- 최근 3개월 임금 입력으로 평균임금 산정
- 퇴직금 총액 계산 및 세부 내역 표시
